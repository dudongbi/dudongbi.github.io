# PWA依赖

```bash
npm install -g @ionic/cli@latest
ionic start --help
ionic start pwa tabs --type vue --capacitor --no-deps --no-git
cd pwa
yarn
yarn upgrade-interactive --latest
npx browserslist@latest --update-db

npm install -g @vue/cli
vue add pwa
```
